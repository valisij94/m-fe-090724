## Практическое занятие №12

### Тема: Методы массивов с колбэком.

Репозитарий содержит материалы по практическому занятию №12, посвященному работе с методами массивов с колбэком в JavaScript.

### Цели занятия
- закрепление знаний, полученных на лекциях;
- практическое применение знаний, полученных на лекциях.

### Задачи на работу с методами массивов
1. пройтись по массиву чисел, для каждого числа вывести, является ли оно четным.
2. пройтись по массиву строк, для каждой строки, если она меньше 4 символов, выводить `short`, иначе - `long`
3. из массива чисел, получить новый массив, который будет содержать квадраты чисел
4. из массива строк, получить новый массив, который будет содержать строки, приведенные к верхнему регистру (метод строк `toUpperCase`).
5. из массива строк получить новый массив, который будет содержать массив ОБЪЕКТОВ, у которых будут свойства:
     * value - сама строка
     * size - если длина строки меньше 4 символов, то `short`, иначе - `long`
6. задача "со звездочкой". Реализовать свой map.
7. получить новый массив, состоящий только из нечетных элементов массива чисел.
8. получить из массива строк новый массив, в котором будут только строки, чья длина больше 4 символов.
9. задача "со звездочкой". Реализовать свой filter.

### Задачи на работу с функциями
1. Пристрелочная. Пишем функцию, которая будет считать сумму двух чисел.
2. Пишем функцию-калькулятор. Она принимает 3 аргумента, первое число, второе число, и символ математической операции. Символ - это "+", "-", "*", "/". В зависимости от этого символа, вернуть соответственно сумму, разность, произведение или частное первого и второго аргументов.
3. Пишем функцию `getCircleParam`. Она принимает 2 аргумента: радиус окружности, и требуемую характеристику (`prop`). Если `prop === 'length'`, вернуть длину окружности (2 * Math.PI * radius). Иначе - вернуть площадь круга. По умолчанию, значения аргументов - 0 и `length`.
```
getCircleParam(5, 'length'); // 31.4
getCircleParam(); // 0
```
4. Пишем функцию `reverseSentence`. Она принимает на вход строку из слов, разделенных пробелами, и должна вернуть эту строку в обратном порядке (то есть поменять порядок слов).
5. Пишем функцию `countVovels`. Она должна принимать строку, и возвращать количество гласных в этой строке. Гласные - `a,o,e,i,u,y`.  Функция должна быть регистронезависимой.
```
countVovels('timon'); // 2
countVovels('PUMBA'); // 2
```
